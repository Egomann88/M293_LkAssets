Box model

box sizing legt ein feste grösse vor, die durch nichts verändert wird.
Beispiel:
  1. width ist 302px gross (2px wegen der Border)
  2. Mit Box-sizing wird alles nach ihnen gerichtet und die 2px "verschwinden"
  3. Jetzt aht die Box 300px als feste grösse und kann nichtmehr verändert werden

Flexbox

Flexbox ist vereinfachtes grid

  • display: flex --> aktiivert den flex (̿▀̿‿ ̿▀̿ ̿)
  • flex-wrap: wrap --> controls how wrapping works
  • flex-direction: row --> direction of content flow
  • justify-content: start --> align along the content flow ( main direction )
  • align-items: auto --> vertical align within a line ( cross direction )
  • align-content: start --> vertical align within container
  • flex-grow: start --> controls the proportion of size growing
  • flex-shrink: start --> controls the proportion of size shrinking

CSS-Grid Terminologie

  • display: grid;
  • grid-template-columns: repeat(5, 1fr) --> Macht 5-Mal das 1fr (abstände gleichmässig verteilen) Horizontal
  • grid-template-rows: 2fr 1fr 2fr --> Macht 2fr Oben und Unten ; 1fr in der Mitte Vertikal
  • grid-column-gap: 15px --> Horizontaler Abstand
  • grid-row-gap: 15px --> Vertikaler Abstand
  • grid-column: 1 / 6 --> Fängt bei "Teil" 1 (ganz Links) an und geht bis "Teil" 6 (ganz rechts)
  • grid-row: 2 / 3 --> Fängt bei Reihe 2 an und geht bis Reihe 3
  • grid-area: 1 / 2 / 3 / 6 --> 1, 2 sind die Anfangswerte ; 3, 6 sind die Schlusswerte (bis dahin gehen sie)
